# Vaishnavi Hadke - Digital Marketing Portfolio

This is a personal portfolio website for Vaishnavi Hadke, a professional digital marketer. It's built with Next.js, React, and Tailwind CSS to showcase her skills, experience, and projects.

## üöÄ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or later is recommended)
- npm, yarn, or pnpm

### Installation

1.  **Clone the repository:**
    ```sh
    git clone <your-repository-url>
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd <project-directory>
    ```
3.  **Install dependencies:**
    ```sh
    npm install
    ```

### Running the Development Server

To start the development server, run the following command:

```sh
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result. You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.
