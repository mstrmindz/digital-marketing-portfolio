# **App Name**: Vaishnavi Hadke Portfolio

## Core Features:

- Homepage: Visually appealing and informative landing page showcasing key skills, experience, and call-to-action buttons.
- About Me: Dedicated page detailing professional background, education, philosophy, and a professional headshot.
- Services Showcase: Detailed presentation of digital marketing services offered, highlighting benefits and key features.
- Portfolio Display: Interactive portfolio displaying case studies, client testimonials, and project highlights with before/after metrics.
- Contact Form: Easy-to-use contact form with validation, direct email link, and social media links for inquiries and networking.
- Theme Customization: Toggle between light, dark, and system-preference themes for enhanced user experience.

## Style Guidelines:

- Primary color: Dusty Rose (#C185A3) – a desaturated pink tone that conveys both professionalism and creativity.
- Background color: Off-White (#F5F2F3) – provides a soft backdrop that keeps the focus on the content.
- Accent color: Lilac (#B39DDB) – offers a modern, tech-savvy contrast, creating highlights and active states.
- Headline font: 'Playfair', a modern serif font giving an elegant, high-end feel
- Body font: 'PT Sans', a humanist sans-serif that pairs nicely with the display font above
- Use Heroicons for a consistent and clean design. Icons should be minimalistic and relate to specific content sections.
- Modern, responsive design using Tailwind CSS, ensuring optimal viewing experience across all devices with a utility-first approach.
- Subtle hover effects, loading states, and smooth transitions between themes for enhanced user engagement.
